Purpose: verified artist + product facts the agent can pull into
generation prompts and captions. Use SPARINGLY — 1–2 details per
prompt, only when they sharpen the concept. The product (the vinyl)
stays the hero; context is seasoning, not the dish.

Refreshed: 2026-07-05 (re-verify anything time-sensitive before use;

## Why context matters right now (the big thread)

Wiz's current moment IS the Cabin Fever era's nostalgia loop. His 2026
mixtape with mgk is literally titled **blog era boyz**, produced by
**Sledgren and ID Labs — the same producers behind Cabin Fever**
(2011). Content angle that writes itself: *"you used to download this
mixtape off a blog in 2011 — now it's pressed on red wax for the first
time ever."* Fifteen years, same producers, physical at last.

## Guardrails (learned the hard way)

- **Never render Wiz himself** — no face/likeness in generations.
  Silhouettes of generic fans/DJs are fine.
- **No weed paraphernalia in prompts** (joints, blunts, smoking
  devices) — fal moderation risk + brand safety. Ambient smoke/haze:
  fine. "4:20" as a text/time gag: borderline, use sparingly.
- **Audio moderation is stricter than visual** (two 4K runs were
  rejected for generated *audio*): avoid prompting chants, crowd
  vocals, or anything singable — request instrumental only, or render
  silent + add mmaudio foley. Don't ask models to *speak/sing* track
  titles like "GangBang"; put titles in text overlays instead.
- Don't fake numbers (chart stats, sales, stream counts) in captions.

## How to use (for the generating agent)

1. Default: product-first prompt, zero context.
2. When a concept needs a hook, pull ONE thread from here: the
   blog-era/download angle, a black-and-yellow accent, a real track
   title as an end card, "first time ever on vinyl".
3. Cite which thread you used in the experiment ledger notes so
   winners can be traced back to the context that made them.
